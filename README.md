## Stadium-management-system

### 项目说明

该项目为体育馆场地管理系统1.0版本，使用原生Javaee+Jsp+Mysql，数据库连接工具c3p0，数据库操作工具Commons，用监听器完成定时任务，过滤器实现安全跳转。
